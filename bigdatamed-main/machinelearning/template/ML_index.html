Analisis Exploratorio de datos

Permite la exploración de los datos, ofreciendo un detallado análisis estadístico. Se incluyen todas las herramientas necesarias para que el usuario pueda llevar a cabo un extenso análisis exploratorio de datos univariable y bivariable. Este análisis es específico en el sentido que, dependiendo el tipo o los tipos de las variables consideradas (si son simbólicas o numéricas), se muestran una serie de figuras, estadísticos y test de hipótesis determinados.

Resultados previos de ejecución

Permite el acceso a los reportes automáticos de ejecuciones anteriores. En este módulo se ofrece una lista con ejecuciones anteriores y se permite acceder a los reportes generados automáticamente que contienen el análisis de resultados de los algoritmos asociados a cada ejecución. Cada tipo de problema tiene una serie de medidas con las que analizar su desempeño y cada una de las medidas obtenidas se describen, con algunas pautas para su interpretación.
