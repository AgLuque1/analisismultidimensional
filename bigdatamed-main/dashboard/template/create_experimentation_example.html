Workflow de trabajo de la aplicación





Operación Detalles
Configuración del Dataset Configuración del dataset con el que va a llevarse a cabo la experimentación.
Explorar datos De forma opcional, se exploran los datos, haciendo posible la formulación de problemas de diversos ámbitos del aprendizaje automático, que pueden resolverse con las herramientas de machine learning de la aplicación.
Extracción de conocimiento Extracción de conocimiento: mediante el uso de las herramientas disponibles se solucionan problemas de machine learning, minería de datos, etc., de forma que se extrae conocimiento de los datos seleccionados como entrada.
Análisis de resultados Con el módulo de histórico de resultados se analizan, de forma personalizada según el problema y el algoritmo empleados, las salidas provistas por los algoritmos y se ofrecen explicaciones que ayudan al usuario a interpretar y comprender los resultados fruto de la experimentación realizada.
